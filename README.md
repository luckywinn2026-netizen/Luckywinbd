# Lucky Win BD

Bangladesh's #1 betting & casino platform – slots, crash games, and more.

## Developer

**Mushfiqur Rahman**

## Tech Stack

- **Frontend**: Vite, TypeScript, React, shadcn-ui, Tailwind CSS
- **Backend**: Node.js (Express)
- **Database**: Supabase (PostgreSQL)
- **Mobile**: Capacitor (Android APK)

## Getting Started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run cap:apk:release:all
```

## Project Structure

- `src/` – React frontend
- `backend/` – Express API
- `android/` – Capacitor Android app
- `supabase/` – Migrations & Edge Functions
